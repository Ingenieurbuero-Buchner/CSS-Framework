Positonierung von DIV-Container

Auf dieser Seite werden die verschiedenen Eigenschaften zur Positionierung von Elementen erklärt, die in CSS verfügbar sind.
  • position: static
  • position: relative
  • position: absolute
  • float

position: static

Standardmäßig ist für alle Elemente die Eigenschaft position mit dem Wert static belegt. Dies bedeutet, dass die Elemente nicht positioniert werden und da erscheinen wo sie normalerweise (laut Reihenfolge im Quelcode) erscheinen würden. Diese Eigenschaft wird nur angegeben, wenn die zuvor angegebene Eigenschaft außer Kraft gesetzt werden soll. 

                #div-1{
                   position: static;
                }

position: relative

Wenn position: relative, dann können auch die Eigenschaften top or bottom und left or right genutzt werden. Hiermit werden die Elemente relativ zu der Position, an der sie normalerweise (laut Reihenfolge im Quellcode) angeordnet sind positioniert. 

                #div-1 {
                   position: relative;
                   top: 20px;
                   left: -40px;
                }

Anmerkung: Der Platz den div-1 vor der Verschiebung belegt hat, ist jetzt ein leerer Platz. Andere Elemente werden durch die Verschiebung nicht beeinflußt.

position: absolut

Wird ein Element mit position: absolute deklariert, dann wird es von der Seite entfernt und genau auf die angegebene Position gesetzt. 

                #div-1a {
                    position: absolute;
                    top: 0;
                    right: 0;
                    width: 200px;
                }

Anmerkung: Ab dem Zeitpunkt andem #div-1a von der Seite entfernt wurde, werden alle anderen Elemente den frei gewordenen Platz belegen, sprich sie werden nachrücken.

position: relative + position:absolute

Wenn ein übergeordnetes Element auf position: relativ gesetzt ist, dann werden alle Elemente innerhalb des untergeordneten div- Containers, die auf position: absolute gesetzt sind, relativ zu diesem übergeordneten Container positioniert. 

               #div-1 {
                   position: relative;
               }
               #div-1a {
                   position: absolute;
                   top: 0;
                   right: 0;
                   width: 200px;
               }

two column absolute

Damit kann man bereits ein Layout mit zwei Spalten generieren. 

                #div-1 {
                    position: relative;
                }
                #div-1a {
                    position: absolute;
                    top: 0;
                    right: 0;
                    width: 200px;
                }
                #div-1b {
                    position: absolute;
                    top: 0;
                    left: 0;
                    width: 200px;
                }

Ein weiterer Vorteil der absoluten Positionierung ist, dass die Elemente in beliebiger Reihenfolge im HTML-Code beschrieben werden können. Dies hat keinen Einfluß auf die Postionierung auf der fertigen Seite.

two column absolute height

Dies ist keine so gute Lösung. Da wir gewöhnlich nicht wissen, wieviel Text in einer Spalte unterkommen soll. 

            #div-1 {
                position: relative;
                height: 250px;
            }
